# 🍳 CookMate - Smart Recipe Finder

CookMate is a smart recipe finder app that helps you discover recipes based on ingredients you have or by searching recipe names.

---

## 🚀 Features

- 🔍 Search recipes by name
- 🥕 Search recipes by ingredients
- 📋 View full recipe details
- 🍲 Ingredients + instructions included
- ⌨️ Press **Enter** to search
- ❌ Press **ESC** to close recipe popup
- 🎯 Clean and responsive UI
- 📱 Mobile-friendly design
